板子选型​

需要考虑的因素：

  1. 需要外接两个CSI摄像头，180°以上
    1. rocketchip kernel维护的摄像头有：
      1. IMX219 这个有很多镜头的，找到了个200度的
      2. OV5647
      3. TC35874X
  2. 4G模块可配usb的
  3. 网卡带wifi+ble的，PCIE2/3
  4. 麦克风先随便买一个i2s的看我能不能驱动
