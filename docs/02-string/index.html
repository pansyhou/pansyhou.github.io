顺序存储​

链式存储​

typedef struct StringNode{
    char ch[4];//一次存4个，比一次存1个的密度高

int StrCpm(SString &s,SString T){

字符串模式匹配​

在主串中找到与模式串相同的子串，并返回其所在位置。

朴素模式​

就是上面的畜生操作，最多匹配n-m+1个

朴素指针模式​

子串匹配失败后

i=i-j+2 从最开始匹配的地方+1

j=1 恢复初始位置

最坏时间复杂度：O(nm)（O((n-m+1)*m)=O(nm)) n-m+1个子串

主串：aaaaaaaaaaab

模式串：aaab
