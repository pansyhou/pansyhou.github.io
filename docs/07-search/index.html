  3. h=[log2(n+1)]向上取整。参考完全二叉树的计算方法

红黑树总结​

根节点黑高为h的红黑树，内部节点数至少有多少个？

answer：总共h层黑节点的满树

为什么一定要满树？ （会破坏黑路同的性质，不对称了，不对称那边就应该不是h）

这种的结点数最少有2h−12^h-12h−1

回答了红黑树h≤2log2(n+1)h\le 2log_2(n+1)h≤2log2​(n+1)

because，若总高度=h，根节点黑高≥h2\ge \frac{h}{2}≥2h​ 由纯黑满树的结点数可得n≥2h2−1n\ge 2^{\frac {h}{2}}-1n≥22h​−1 化简就能得到上式
